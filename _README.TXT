
# Install

Open with:
https://gitpod.io/#https://github.com/aknip/hardhat-tests

1. Terminal:
npm install
npx hardhat node

2. Terminal:
npx hardhat --network localhost run scripts/deploy.js
cd frontend
npm install
npm run start

Browser: 
http://localhost:3000/
